# OverlayTimer 공통 운영 메모

## 목적

`DPS_SPEC.md`나 타이머 기능에 공통으로 적용되는 운영/설정 규칙을 한 곳에 정리한다.

---

## 트레이 아이콘 메뉴

| 항목 | 동작 |
|---|---|
| 타이머 최소화 / 타이머 복원 | 타이머 오버레이 창 최소화/복원 |
| DPS 최소화 / DPS 복원 | DPS 오버레이 창 최소화/복원 |
| 종료 | 앱 종료 |

- 메뉴를 열 때 현재 창 상태를 반영해 메뉴 텍스트를 갱신한다.
- 최소화해도 작업 표시줄 버튼은 유지된다.

---

## 오버레이 공통 입력

- `Ctrl + 좌클릭 드래그`: 오버레이 창 이동
- `Ctrl` 해제: 클릭 투과 모드 복귀

---

## 설정 파일 포인트

### `config.json`

- `packetTypes`: `buffStart`, `buffEnd`, `enterWorld`, `dpsAttack`, `dpsDamage`
- `overlays`: `timer`와 `dps` 각각 `enabled`, `x`, `y` 지원

### `skill_names.json`

- 형식: `key1(uint) -> 스킬 이름` 매핑 테이블
- JSON 키는 문자열 숫자 사용
- 매핑이 없으면 UI에서 숫자 ID 그대로 표시
- 스킬명 표기 규칙: `직업-스킬` (예: `대검-평타`)

예시:

```json
{
  "1215289592": "대검-평타",
  "98765432": "대검-회전베기"
}
```
